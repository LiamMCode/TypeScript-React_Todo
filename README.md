# TypeScript-React_Todo
 An updated version of https://github.com/LiamMCode/React_TODO which uses TypeScript rather than JavaScript
    - Implements webpack, Typescript and React

- `webpack`
- `webpack-cli@3.3.12`
- `webpack-dev-server`
- `typescript`
- `@types/react` 
- `@types/react-dom`
- `ts-loader`
- `@types/react-router-dom`
